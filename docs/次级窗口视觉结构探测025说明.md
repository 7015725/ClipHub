# ClipHub 次级窗口视觉结构探测 025

## 目的

验证阶段 3D1 第一部分：搜索与筛选、新增编辑、标签管理三个次级窗口的视觉结构优化及功能回归。

```text
moduleSetVersion=20260722.20
probeVersion=1
```

## 本轮生产变更

### Editor / Tag Editor

- 深色和浅色背景改为完全不透明；
- 增加 `FLAG_DIM_BEHIND`，背景暗化量为 `0.72`；
- 移除 `FLAG_NOT_TOUCH_MODAL`，次级窗口打开期间不允许误触后方主列表；
- “关闭”文字按钮改为紧凑 `×`；
- 新增与编辑窗口最大高度约 430dp；
- 标签窗口根据标签数量在 310–492dp 内动态调整；
- 输入区、标题、说明和操作按钮降低字号、字重与内边距；
- 标签选中态降低饱和度；
- 标签创建或删除后重新计算窗口高度。

### Filter

- 使用与 Editor 相同的不透明模态窗口；
- 窗口根据实际来源、类型和标签分组在 320–438dp 内调整；
- 空分组不再占用空间；
- 芯片字号、间距与内边距缩小；
- 横向芯片列表启用渐隐边缘提示；
- 关闭入口统一为 `×`；
- 保留原搜索、组合筛选、重置和事件重应用逻辑。

## 不在本轮范围

以下内容保持 `20260722.19` 已验证实现：

- 主列表卡片和操作区；
- 复制、删除、撤销、置顶和排序；
- 长文本详情窗口；
- Repository、数据库 schema 和事件格式。

详情双层容器和主列表操作密度将在 3D1 后续小阶段单独处理。

## 自动探测范围

025 使用隔离目录：

```text
shortx.getShortXDir()/ClipHubProbe025/
```

探测会：

1. 停止正式实例并启动隔离实例；
2. 停止隔离剪贴板监听；
3. 写入三条记录及两个标签；
4. 打开暗色主列表；
5. 验证 Filter 的模态、不透明、尺寸和中英文搜索；
6. 验证新增窗口并实际保存一条记录；
7. 验证两标签窗口的紧凑高度；
8. 新建第三标签，确认窗口高度动态增加；
9. 解绑并删除第三标签；
10. 保持编辑窗口打开后执行 App.stop，验证清理；
11. 重启隔离实例，验证记录与标签持久化；
12. 恢复正式实例并删除隔离目录。

不会读取或替换用户系统剪贴板，不会修改正式数据库。

## 执行顺序

当前正式实例仍运行 `.19`，需要先更新到 `.20`。

### 1. 同步仓库

```sh
cd ~/projects/github/ClipHub
git fetch origin
git switch agent/initialize-project-skeleton
git pull --ff-only
node --check src/ch_10_editor.js
node --check src/ch_11_filter.js
node --check probes/cliphub_secondary_visual_probe_025.js
python3 scripts/validate_es5.py .
```

```sh
grep -n "moduleSetVersion" module-manifest.json
grep -n "probeVersion" probes/cliphub_secondary_visual_probe_025.js
```

应显示：

```text
20260722.20

```

### 2. 复制探测文件

```sh
cp probes/cliphub_secondary_visual_probe_025.js \
  "/storage/emulated/0/Download/ClipHub_次级窗口视觉结构探测_025.js"
```

### 3. 更新正式模块

执行已有停止任务，然后运行完整 `ClipHub.js`。
