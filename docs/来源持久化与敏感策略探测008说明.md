# ClipHub 来源持久化与敏感策略探测 008

## 1. 当前版本

```text
moduleSetVersion=20260721.8
```

## 2. 实现内容

模块集 `20260721.8` 修改：

- `ch_03_database.js`
  - schema 从 1 升级为 2；
  - `clipboard_items` 增加 `is_sensitive INTEGER NOT NULL DEFAULT 0`；
  - 增加敏感记录索引。
- `ch_06_repository.js`
  - 插入和更新支持 `is_sensitive`；
  - 查询支持 `sensitiveOnly` 与 `excludeSensitive`。
- `ch_04_clipboard.js`
  - 读取 `getPrimaryClipSource()`；
  - 解析来源应用名称和 UID；
  - 来源解析成功时写入 `source_confidence=100`；
  - 默认 `sensitivePolicy=skip`；
  - 可配置 `sensitivePolicy=save`；
  - 支持 `ignorePackages`；
  - 日志不记录剪贴板正文。

## 7. 默认隐私策略

当前默认：

```text
sensitivePolicy=skip
```

带标准敏感标记的内容不读取正文、不计算正文哈希、不写入数据库。事件只记录忽略原因、来源包名和线程信息。

后续设置页可提供：

- 跳过敏感内容；
- 保存并标记敏感内容。

默认值保持“跳过”。
