# ClipHub 次级窗口截图复核 026

## 目的

对模块集 `20260722.20` 中已经通过自动回归的三个次级窗口进行人工视觉复核：

1. 搜索与筛选；
2. 新增编辑；
3. 标签管理。

026 不修改生产模块、manifest、正式数据库或系统剪贴板。

```text
moduleSetVersion=20260722.20
probeVersion=1
sceneCount=3
sceneDurationMs=10000
```

## 场景

### 1/3 搜索与筛选窗口

检查：

- 窗口是否真正不透明；
- 后方主列表是否被适度暗化；
- `×` 关闭入口是否紧凑；
- 搜索框、分组标题、芯片和重置按钮是否协调；
- 长标签或长来源是否仍存在难以理解的截断；
- 横向渐隐边缘是否能够提示继续滑动；
- 窗口高度是否明显减少无效空白。

### 2/3 新增编辑窗口

检查：

- 输入框高度与窗口高度是否合理；
- 标题、说明、输入框和保存按钮的间距；
- `×` 和保存按钮是否形成清晰的主次关系；
- 输入法出现后是否遮挡保存按钮；
- 后方主列表是否不再干扰阅读。

### 3/3 标签管理窗口

检查：

- 两标签时窗口是否紧凑；
- 已绑定标签的选中态是否降低饱和度；
- 长标签名称和删除按钮是否都能完整显示；
- 新建标签输入区与标签列表的间距；
- 后方内容是否被充分暗化。
