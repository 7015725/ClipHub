# 设置页作者与博客入口

## 目标

在设置页“数据与关于”卡片中，将运行状态和作者信息分为两个层级：

```text
数据与关于
当前数据库、模块和项目相关信息

运行信息
剪贴板记录：…
标签数量：…
数据库：…
Schema：v2
模块集：…

关于作者
林深见鹿
个人博客                         xin-blog.com ↗
```

## 交互

- “个人博客”整行可点击；
- 点击时有透明度按压反馈；
- 使用系统 `Intent.ACTION_VIEW` 打开 `https://xin-blog.com`；
- 使用 `FLAG_ACTIVITY_NEW_TASK`，适配 ShortX/system_server 运行环境；
- 打开失败时不让异常逃逸到系统主线程，错误写入 Settings 状态。

## 版本

```text
Settings MODULE_VERSION=12
moduleSetVersion=20260723.11
```

## 修改边界

只修改：

```text
src/ch_13_settings.js
module-manifest.json
```

不修改：

- SQLite schema 和设置数据；
- 标签管理及事务；
- 翻译配置；
- 输入法避让和滚动锚点；
- 后台入口与悬浮窗开关；
- Navigation 和其他页面。

## 可探测状态

Settings 状态新增：

```text
blogLinkPresent
blogOpenCount
blogOpenSuccessCount
lastOpenedUrl
```

并提供：

```text
performOpenBlogClick()
```

用于后续真机探测验证博客入口存在、点击请求成功以及 URL 正确。
