# ClipHub 阶段 3D2：新 UI 全量复刻方案

## 1. 阶段目标

阶段 3D2 的目标不是继续审查旧 UI，也不是只做宽度与安全区探测，而是依据用户提供的 5 张 Android 端参考图，对 ClipHub 的主要页面进行完整视觉与交互复刻。

参考页面：

1. 首页 / 列表页；
2. 搜索 / 筛选页；
3. 新增 / 编辑页；
4. 详情 / 识别 / 翻译页；
5. 标签 / 分类管理页。

复刻目标：

- Android 手机端优先；
- 视觉结构、层级、间距、圆角、色彩、按钮位置和卡片密度按参考图统一；
- 交互流程与参考图保持一致；
- 已有数据、剪贴板监听、SQLite、系统返回和最近任务退出能力继续保留；
- Rhino ES5 严格兼容；
- 不增加 WebView；
- 不使用自定义底部系统手势拦截层。

## 2. 总体视觉框架

所有页面使用统一的悬浮底板结构：

```text
当前应用界面
└─ 半透明遮暗背景
   └─ ClipHub 浮层面板
      ├─ 顶部拖动手柄
      ├─ 页面标题栏
      ├─ 页面主体滚动区
      └─ 固定底部操作区（按页面需要显示）
```

统一视觉特征：

- 白色或极浅紫灰面板；
- 大圆角浮层；
- 紫色作为主强调色；
- 轻量描边与低强度阴影；
- 卡片圆角统一；
- 图标使用单色线性风格；
- 标签使用浅色胶囊背景；
- 主操作使用紫色实心按钮；
- 次操作使用浅紫或白色描边按钮；
- 正文、次级信息和说明文字形成明确三级层级。

## 3. 页面 1：首页 / 列表页

### 3.1 页面结构

```text
拖动手柄
标题栏：图标 + 全局剪切板 + 置顶 + 设置
搜索行：搜索框 + 筛选 + 新增
状态行：总条数 + 排序提示 + 管理入口
剪切板卡片列表
底部快捷操作栏：置顶 / 编辑 / 新增 / 删除 / 翻译
```

### 3.2 卡片结构

每张卡片包含：

- 来源应用图标；
- 内容摘要；
- 时间；
- 内容类型标签；
- 自定义标签；
- 收藏 / 置顶状态；
- 更多操作入口；
- 选择状态；
- 长按拖动排序能力。

### 3.3 交互

- 点击卡片进入详情页；
- 长按卡片进入选择状态；
- 更多按钮承担标签入口和拖动排序；
- 点击星标切换置顶；
- 管理模式下底部操作栏作用于选中项；
- 搜索框进入搜索 / 筛选页；
- 筛选按钮进入高级筛选；
- 新增按钮进入新增页；
- 旧程序化 `performItemClick()` 仍保留复制语义，用于兼容既有探测和调用方。

### 3.4 当前进度

第一轮模块集 `.26` 已通过探测 033 的结构与生命周期检查，但真机截图发现面板比例、卡片密度、来源图标、更多按钮、选择语义和底栏样式仍不符合参考图。

第二轮模块集 `.27` 已完成：

- 面板目标尺寸调整到接近 `390 × 720dp`；
- 遮暗值调整为 `0.44`；
- 优先加载来源应用图标；
- 卡片右侧改为时间 + 星标 / 更多横向布局；
- 普通浏览状态不再预留选择圆点；
- 点击卡片进入详情，长按进入选择；
- 选择不再触发复制；
- 中央新增按钮强化，未选中操作弱化。

当前等待探测 034 真机截图确认首页第二轮视觉基线。

## 4. 页面 2：搜索 / 筛选页

### 4.1 基础搜索

- 关键词输入；
- 实时搜索；
- 搜索历史；
- 清除关键词；
- 结果条数；
- 关键词高亮；
- 排序方式切换。

### 4.2 高级筛选面板

筛选项：

- 关键词；
- 来源应用，多选；
- 内容类型，多选；
- 标签，多选；
- 排序方式；
- 重置；
- 应用筛选。

高级筛选采用面板内抽屉 / 浮层结构，不跳出 ClipHub 主面板。

## 5. 页面 3：新增 / 编辑页

### 5.1 页面结构

```text
拖动手柄
标题栏：页面状态 + 关闭
标题输入
备注输入
正文多行输入 + 字数统计
智能识别结果与类型修正
来源应用选择
标签管理
置顶开关
收藏开关
翻译预览
智能识别预览
从剪贴板粘贴 / 快速清空
取消 / 保存
```

### 5.2 功能边界

- 新增和编辑共用一套页面骨架；
- 编辑时回填已有标题、备注、正文、来源、标签、置顶和收藏状态；
- 保存前进行内容长度、空内容和字段合法性校验；
- 系统返回或关闭按钮不自动保存；
- 保存后回到首页并刷新对应卡片；
- 保留现有 SQLite 与 EventBus 数据链。

## 6. 页面 4：详情 / 识别 / 翻译页

### 6.1 页面结构

```text
顶部摘要：来源图标 + 内容摘要 + 时间 + 收藏
原始内容卡片
智能识别区域
一键翻译区域
快捷操作区域
来源与创建时间信息
```

### 6.2 识别能力

按内容展示对应操作：

- 网址：打开链接；
- 电话：拨号；
- 邮箱：发送邮件；
- 代码：复制代码；
- 普通文本：复制、搜索、翻译、分享、朗读。

### 6.3 翻译能力

- 源语言选择；
- 目标语言选择；
- 语言互换；
- 翻译结果；
- 复制翻译结果；
- 保留已有翻译配置与调用边界。

## 7. 页面 5：标签 / 分类管理页

### 7.1 页面结构

```text
拖动手柄
标题栏：标签 / 分类管理 + 置顶 + 设置
标签概览统计卡片
系统标签列表
自定义标签列表
新建标签
批量归类预览
底部操作栏：批量归类 / 移动 / 移除标签 / 完成
```

### 7.2 功能

- 标签数量统计；
- 系统标签不可删除；
- 自定义标签新建、重命名和删除；
- 标签颜色；
- 标签拖动排序；
- 多选记录批量添加或移除标签；
- 卡片预览；
- 数据继续使用现有标签表与关联表。

## 8. 模块映射

| 模块 | 3D2 主要职责 |
|---|---|
| `ch_07_theme.js` | 新视觉令牌、紫色主题、文本层级、卡片与按钮背景 |
| `ch_08_window.js` | 统一浮层底板、拖动手柄、遮暗层、尺寸与安全区 |
| `ch_09_list.js` | 首页、列表卡片、选择模式、底部快捷栏、详情页入口 |
| `ch_10_editor.js` | 新增 / 编辑页、标签编辑相关表单 |
| `ch_11_filter.js` | 搜索、筛选、高级筛选面板 |
| `ch_12_translation.js` | 详情页导航边界、识别与翻译区域协作 |
| `ch_13_settings.js` | 与新 UI 对应的设置入口与视觉参数 |
| `ch_14_event_bus.js` | 页面刷新与数据变更通知，不改变最近任务观察器 |
| `ch_15_app.js` | 页面入口协调与统一显示 / 隐藏 |

不在本阶段增加 WebView，不将页面拆成独立 Android Activity。
