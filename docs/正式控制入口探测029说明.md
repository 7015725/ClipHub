# ClipHub 正式控制入口探测 029

## 目的

验证模块集 `20260722.22` 的正式用户入口和动态控制协议：

- 控制端点 schema v2；
- `show / hide / toggle / status / stop` 五个命令；
- 主线程回调；
- 主列表打开和刷新；
- 筛选、详情等次级窗口统一关闭；
- 回执正文脱敏；
- 旧停止命令兼容；
- 控制协议跨重启保持。

## 隔离范围

```text
shortx.getShortXDir()/ClipHubProbe029/
```

探测不会修改正式数据库，不会读取或替换系统剪贴板。隔离实例启动后会立即停止剪贴板监听并写入三条固定记录。

```text
moduleSetVersion=20260722.22

```

## 自动流程

1. 使用现有 `stop` 协议停止正式实例；
2. 启动隔离实例；
3. 核对控制端点命令集合；
4. `status` 验证初始无 UI；
5. `show` 打开三条历史记录；
6. `toggle` 关闭并再次打开；
7. 打开筛选窗口后执行 `hide`，验证全部关闭；
8. 打开长文本详情后执行 `toggle`，验证详情和主列表均关闭；
9. 再次 `show`；
10. 检查全部回执不包含 `content`；
11. 使用同一协议执行 `stop`，验证旧停止语义；
12. 重启隔离实例，再次验证 `show/hide/stop`；
13. 恢复正式实例并删除隔离目录。
