要求模块集：

```text
moduleSetVersion=20260723.04

```

## 验证范围

1. 设置页采用 `reference_settings_v2`；
2. 标签名称、颜色预览、关联数量和拖动手柄正常；
3. 标签排序结果持久化到 `manual_order`；
4. 删除操作需要二次确认；
5. 删除标签只解除关联，不删除剪贴板记录；
6. 编辑页标签选择器采用 `reference_tag_selector_v1`；
7. 标签选择器取消或系统返回不写入 `clipboard_item_tags`；
8. 点击标签选择器“完成”后，仍需点击正文编辑页“保存”才提交关联；
9. 编辑已有记录与新增记录均支持多标签；
10. Filter Root 卡片使用标签自定义颜色，并继续显示两个标签和 `+N`；
11. Settings 返回、Editor 两层返回和唯一 Filter Root 保持正常；
12. 数据库 schema v2、Navigation v3、正式实例恢复和运行锁清理保持不变。
