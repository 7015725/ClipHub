# ClipHub 新首页 UI 探测 033

## 目标

探测模块集 `20260722.26` 的共享视觉基础和首页 / 列表页首版实现，并生成一张用于与参考图页面 1 对照的真机截图。

本探测不验证其余四个页面，也不代表首页已经达到最终像素级复刻标准。


## 已实现视觉结构

### 共享主题

- 亮色和暗色紫色主题；
- 页面底板、卡片、胶囊标签、描边和文字层级；
- 危险、成功、警告、网址、电话、邮箱和代码类型色；
- 统一圆角、间距、拖动手柄和底部操作栏尺寸。

### 浮层窗口

- 大圆角底部浮层；
- 顶部居中拖动手柄；
- 背景遮暗；
- 默认底部居中；
- 保留拖动、固定、折叠、位置保存和安全区夹紧；
- 保留系统返回、最近任务退出和 WindowManager 清理链。

### 首页 / 列表页

- 标题、置顶和设置入口；
- 搜索框、筛选按钮和新增按钮；
- 条目数量、排序提示和管理入口；
- 来源 / 类型图标、内容摘要、时间、类型标签和自定义标签；
- 星标和更多 / 拖动排序入口；
- 选中卡片描边和勾选状态；
- 固定底部操作栏：置顶、编辑、新增、删除、翻译。

当前“翻译”按钮暂时进入详情页，完整识别与翻译界面在阶段 3D2-4 实现。

## 运行步骤

1. 在 Termux 同步 `agent/initialize-project-skeleton`；
2. 运行 `ClipHub.js`，确认同步到模块集 `.26`；
3. 将 `probes/cliphub_new_home_ui_probe_033.js` 的完整内容复制到新的 ShortX JavaScript 任务；
4. 建议任务名：`ClipHub 新首页UI探测033`；
5. 运行探测；
6. Toast 出现“新首页 / 列表页首版 · 请截图”后截取一张未裁剪原始截图；
7. 等待约 18 秒，回传完整 JSON 和截图。

## 边界

- 使用隔离数据库，不修改正式剪贴板历史；
- 正式实例原本运行时，探测结束后自动恢复；
- 正式实例原本未运行时，探测结束后保持未运行；
- 不新增底部系统手势拦截层；
- 不停止正式版本后续的后台剪贴板监听；
- PR 继续保持 Draft，未合并 `main`。
