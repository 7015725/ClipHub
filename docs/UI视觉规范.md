# ClipHub UI 视觉规范

## 1. 适用范围

本规范用于复刻五张 Android 全局剪切板参考图中的手机界面部分。

运行时不复刻：

- 手机机身；
- 图片外围编号、虚线和说明文字；
- 顶部宣传标题和 Android Only 标记；
- 底部模块说明；
- 参考图背景中的聊天应用内容。

ClipHub 实际运行时通过 `WindowManager` 显示在当前应用上方，并使用窗口 `dimAmount` 完成背景遮暗。

## 2. 视觉模式

首轮视觉验收固定使用 `reference_light` 模式：

- 不读取系统动态颜色；
- 不使用 OEM 默认按钮背景；
- 不使用系统默认 Switch 外观；
- 图标由项目内 Drawable 自绘；
- 所有颜色、圆角、字号和间距集中在 `src/ch_07_theme.js`。

系统动态颜色和暗色主题在参考图视觉验收通过后再接入。

## 3. 核心颜色

| 令牌 | 色值 | 用途 |
|---|---|---|
| `primary` | `#6647D9` | 主按钮、选中状态、强调图标 |
| `primaryDark` | `#5638C8` | 主色按压或深色强调 |
| `primarySoft` | `#EEE9FF` | 筛选按钮、圆形按钮、选中容器 |
| `primaryFaint` | `#F7F4FF` | 底部操作栏、弱强调背景 |
| `panel` | `#FBFAFF` | 整体悬浮面板背景 |
| `surface` | `#FFFFFF` | 卡片和输入框背景 |
| `textPrimary` | `#24212D` | 标题和正文 |
| `textSecondary` | `#777281` | 时间、说明和次级文字 |
| `textTertiary` | `#A19BAA` | 占位文字和未选图标 |
| `border` | `#E9E5F0` | 普通卡片边框 |
| `selectedBorder` | `#7659DF` | 选中卡片描边 |

## 4. 尺寸基线

| 项目 | 基线 |
|---|---:|
| 面板左右安全间距 | 12 dp |
| 面板圆角 | 24 dp |
| 卡片圆角 | 14 dp |
| 输入框圆角 | 18 dp |
| 底部操作栏圆角 | 18 dp |
| 标题栏高度 | 54 dp |
| 搜索栏高度 | 48 dp |
| 底部操作栏高度 | 70 dp |
| 卡片间距 | 8 dp |
| 拖动手柄 | 42 × 4 dp |

手机环境中面板默认高度为屏幕高度约 79%，默认靠近屏幕底部。窗口允许通过顶部手柄拖动，并限制在安全显示区域内。

## 5. 字体

- 默认字体：Android `sans-serif`；
- 标题和重要操作：`sans-serif-medium`；
- 代码内容：`Typeface.MONOSPACE`；
- 不使用 Emoji 充当功能图标；
- Emoji 仅允许作为剪切板正文内容展示。

## 6. 图标

功能图标统一由 `src/ch_08_icons.js` 的自绘 `Drawable` 提供：

- 剪切板；
- 搜索；
- 筛选；
- 新增；
- 置顶；
- 设置；
- 收藏；
- 更多；
- 编辑；
- 删除；
- 翻译；
- 返回；
- 标签；
- 复制。

来源应用图标优先使用 `PackageManager.getApplicationIcon()`，应用不存在时使用固定颜色的文字占位图标。

## 7. 页面结构

统一使用一个 WindowManager 根窗口，内部路由切换五类页面：

1. `home`：首页 / 列表页；
2. `filter`：搜索 / 筛选页；
3. `editor`：新增 / 编辑页；
4. `detail`：详情 / 识别 / 翻译页；
5. `tags`：标签 / 分类管理页。

禁止为每个页面创建独立 WindowManager 窗口。

## 8. 动画边界

允许：

- 窗口首次打开时 180 ms 淡入和轻微上移；
- 内部页面切换时 120 ms 淡入和轻微横移；
- 后续列表新增、删除和排序时使用短时状态动画。

禁止：

- 无状态变化时持续刷新；
- Canvas 空转动画；
- 搜索输入时重建整个窗口；
- 页面关闭后仍保留动画回调；
- WindowManager 移除前不取消动画。

## 9. 第一阶段视觉验收

第一阶段只验收首页 / 列表页视觉基线：

- 标题为“全局剪切板”；
- 搜索框、筛选按钮和新增按钮结构完整；
- 显示“共 23 条”；
- 至少展示七条演示卡片；
- 电话卡片处于选中和收藏状态；
- 底部显示置顶、编辑、新增、删除、翻译五项操作；
- 窗口可拖动、返回和关闭；
- 重复执行入口不会创建多个窗口。

## 10. 像素差异边界

| 项目 | 允许偏差 |
|---|---:|
| 核心组件位置 | ±1 dp |
| 文字基线 | ±2 px |
| 圆角和边框 | ±1 px |
| 主要颜色 RGB | 每通道不超过 6 |
| 非文字区域像素差 | 不超过 1.5% |

字体抗锯齿、系统应用图标和不同设备字体渲染允许存在不可避免的像素差异，但页面结构、尺寸、颜色和交互项必须一致。
