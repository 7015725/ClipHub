# ClipHub 交互规范

## 1. 页面结构

悬浮窗分为：

1. 拖动标题栏；
2. 搜索区；
3. 快速类型过滤；
4. 剪贴板列表；
5. 新增入口；
6. 编辑、筛选和标签等二级页面。

## 2. 标题栏

包含：

- 拖动手柄；
- 标题“全局剪贴板”；
- 置顶；
- 收起；
- 关闭。

### 拖动规则

- 标题栏空白区域可拖动；
- 按钮命中区域只执行按钮操作；
- 按下后移动超过阈值才进入拖动；
- 拖动结束后保存位置；
- 位置必须限制在安全显示区域；
- 窗口不能完全移出屏幕。

## 3. 置顶

“置顶”指悬浮窗保持显示，不是剪贴板项目置顶。

状态建议：

- 未置顶：复制或打开目标内容后可以按设置自动收起；
- 已置顶：保持当前窗口；
- 图标必须清楚区分两种状态；
- 状态需要持久化。

剪贴板项目置顶应使用卡片内独立操作，避免概念混淆。

## 4. 搜索栏

- 输入后延迟触发查询；
- 清空按钮仅在有文字时显示；
- 搜索时保留当前过滤条件；
- 搜索结果数量可在筛选入口附近显示；
- 关闭再打开时是否保留搜索词由设置决定。

## 5. 来源应用过滤

应用列表至少显示：

- 应用图标；
- 应用名；
- 包名；
- 当前记录数量。

支持：

- 单选；
- 多选预留；
- “来源未知”；
- 快速清除。

## 6. 类型过滤

首版提供：

- 全部；
- 文本；
- 网址；
- 电话。

选中状态必须同时通过颜色、描边或图标表达，不能只依赖细微色差。

## 7. 剪贴板卡片

建议内容：

```text
拖动手柄
来源应用图标
来源应用名称
复制时间
内容预览
识别类型
自定义标签
置顶状态
更多操作
```

### 点击行为

- 单击卡片：复制原始内容；
- 点击内容区域：可配置为复制或打开详情；
- 长按：进入多选模式；
- 拖动手柄：调整顺序；
- 更多按钮：编辑、翻译、标签、删除。

为保持卡片紧凑，首版不建议同时常驻过多文字按钮。

## 8. 内容预览

- 最多显示若干行；
- 超长文本截断但不修改原始内容；
- 换行应正常展示；
- 命令片段可使用等宽字体；
- Emoji 必须能够正常显示；
- URL 与电话可使用强调样式；
- 强调样式不能影响文本复制结果。

## 9. 编辑器

字段：

- 内容；
- 内容类型；
- 标签；
- 置顶；
- 保存；
- 取消。

规则：

- 内容为空时不允许保存；
- 离开前存在未保存修改时提示；
- 保存按钮防重复点击；
- 编辑器关闭后恢复主窗口焦点和触摸状态；
- 软键盘弹出时确保操作按钮可见。

## 10. 新增

新增项目与编辑共用页面，但需要区分：

- 新增没有来源应用；
- 创建时间为保存时间；
- 可以直接标记标签；
- 保存后是否写入系统剪贴板由用户选择。

## 11. 翻译

- 原文只读展示；
- 译文与原文分离；
- 支持复制译文；
- 保存译文时创建新项目或覆盖当前内容必须明确选择；
- 网络失败不影响原记录；
- 翻译配置不在列表卡片中展开。

## 12. 删除

建议优先使用：

```text
已删除，可撤销
```

而不是每次弹出确认框。

以下情况需要确认：

- 批量删除；
- 清空全部；
- 删除标签并解除大量关联；
- 永久清理回收记录。

## 13. 收起状态

收起后可显示：

- 一个紧凑胶囊；
- 剪贴板图标；
- 未读或新增数量；
- 最近内容的简短提示，可由隐私设置关闭。

收起入口仍可拖动，但需要与点击展开区分。

## 14. 空状态

### 无记录

```text
尚无剪贴板记录
复制一段文本后会显示在这里
```

### 搜索无结果

```text
没有匹配“关键词”的记录
```

### 过滤无结果

```text
当前过滤条件下没有记录
清除过滤条件
```

### 错误状态

```text
剪贴板记录加载失败
重试
```

## 15. 亮暗色

- 使用统一主题模块；
- 文字与背景保证足够对比；
- 删除等危险操作不能只依赖红色；
- 标签颜色需要经过明暗色对比度修正；
- 系统主题变化时单次刷新，不重复重建窗口。

## 16. 动画

允许：

- 打开与关闭淡入淡出；
- 收起与展开尺寸过渡；
- 新增项目轻量进入；
- 删除项目收缩；
- 拖动排序位移动画。

限制：

- 无状态变化时停帧；
- 不使用持续空转动画；
- 动画结束后清理回调；
- WindowManager 移除前终止仍在运行的动画；
- 搜索输入时不对整个窗口执行动画。
